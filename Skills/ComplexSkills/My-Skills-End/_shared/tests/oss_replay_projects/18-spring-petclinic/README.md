# spring-projects/spring-petclinic OSS replay adapter

This adapter is for a real local checkout of `spring-projects/spring-petclinic`. It intentionally does not bundle third-party source code. Set environment variable `OSS_REPLAY_18_PATH` to the local checkout and run `_shared/tests/oss_replay/oss_replay_runner.py`.

Promotion is blocked unless local checkout, commit hash, exact assertions and parser readiness are recorded.
