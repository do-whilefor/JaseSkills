# 不可报告原因清单

1. 目标不在本机、靶场、本地开源项目或明确授权范围。
2. 命中 SRC / 众测 / 应急响应中心禁止边界。
3. 只有工具告警，没有代码证据、动态证据、影响证据。
4. 只有错误页面、异常响应、版本信息或 README 暴露，无法证明实际安全影响。
5. 受影响版本不匹配。
6. 关键配置条件不满足。
7. 权限、角色、租户、数据状态等前置条件不满足。
8. 不能三次稳定复现。
9. 影响证明需要读取真实敏感数据。
10. 验证会造成 DoS、压测、数据破坏、批量探测、权限维持或横向移动。
11. 来源只有 PoC 仓库、截图、二次转载或 AI 摘要。
12. 官方来源和社区来源存在冲突，尚未人工确认。
13. 漏洞类型属于平台不收范围。
14. 修复建议不明确。
15. 证据链无法追溯到来源、代码、请求响应、日志或用户授权输入。
