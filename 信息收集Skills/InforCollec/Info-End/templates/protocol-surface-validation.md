# 协议差异非破坏验证模板

| 协议/入口 | 非破坏动作 | 记录字段 | 禁止动作 | 回流 |
|---|---|---|---|---|
| HTTP 页面 | GET/HEAD/OPTIONS、Accept 差异、尾斜杠差异 | 状态码、头、大小、标题、敏感字段摘要 | 目录爆破、高频访问 | 04/07 |
| JSON API | GET/HEAD/OPTIONS、错误 Accept、缺少非必填参数 | 状态码、Content-Type、字段名、错误体摘要 | 修改数据、危险 payload | 04/07 |
| WebSocket | 只做握手可达性、协议头、认证要求记录 | 状态、路径、是否需要认证、错误摘要 | 长时间订阅、大量消息、发送业务修改消息 | 04/07 |
| SSE | GET 建连后短时间读取头部和首条事件摘要 | Content-Type、事件字段、认证要求 | 长时间保持连接、高频连接 | 04/07 |
| gRPC/RPC | 仅从 proto/schema、反射暴露状态、错误响应摘要判断 | 服务名、方法名、错误摘要 | 暴力枚举、调用修改方法 | 03/04/07 |
| 文件下载 | HEAD/GET 小样本、Content-Disposition、大小、类型 | 文件名、扩展名、大小、缓存头 | 批量下载、越权猜测 ID | 04/07 |
| metrics/health | GET/HEAD、认证状态、字段摘要 | 指标名摘要、版本/路径提示 | 高频拉取、压测 | 04/07 |

不可报告条件：只有协议可达、只对管理员预期可见、工具缺失未执行、只有错误码没有内容、缺少影响证明。
