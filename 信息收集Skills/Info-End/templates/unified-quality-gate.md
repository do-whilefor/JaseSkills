# 统一质量门禁模板

用于所有 Skill 输出前的最后检查。

- 结论是否可交付：是 / 否 / 阶段性可交付
- 不可交付原因：
- 已满足条件：
- 未满足条件：
- 证据来源：文件 / 代码 / 配置 / 运行态请求 / 响应摘要 / 浏览器证据 / 角色差分 / 反思记录
- 文档映射：原文章节 / 能力指纹 / 延伸能力
- 风险等级：高 / 中 / 低 / 信息 / 待确认 / 不可报告
- 需要人工确认：是 / 否；原因：
- 下一步动作：最小安全验证 / 回流某 Skill / 降级待确认 / 不可报告 / 阻断交付

硬规则：

1. 无动态证据，不得输出确定发现。
2. 无文件真实性验证，不得声称文件、脚本、模板存在。
3. 无执行记录，不得声称脚本已执行。
4. 无角色账号，不得输出角色差异结论。
5. 任一敏感值未脱敏，停止交付并重写。
6. 任一冲突未记录，停止最终交付，只能输出阶段性报告。
7. 任一延伸能力未标记来源，停止最终交付。
8. 任一关键 QG 为 UNKNOWN，降级待确认。
