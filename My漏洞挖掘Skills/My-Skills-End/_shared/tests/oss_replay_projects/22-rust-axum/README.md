# tokio-rs/axum OSS replay adapter

This adapter is for a real local checkout of `tokio-rs/axum`. It intentionally does not bundle third-party source code. Set environment variable `OSS_REPLAY_22_PATH` to the local checkout and run `_shared/tests/oss_replay/oss_replay_runner.py`.

Promotion is blocked unless local checkout, commit hash, exact assertions and parser readiness are recorded.
