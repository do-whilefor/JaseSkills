## 13. 仍可能漏掉严重注入类缺陷的具体原因

- 队列 worker payload schema 未拿到，无法判断异步二阶解释器路径。
- 管理员预览需要人工动作，未采集 DOM/响应/日志。
- sourcemap 不存在，隐藏参数只能从压缩 bundle 低置信提取。
- 搜索索引任务未触发，无法验证先存储后索引的搜索 DSL 路径。
- 租户 B 或 API token 身份不可用，无法验证跨租户或 token 与 Web session 差异。
