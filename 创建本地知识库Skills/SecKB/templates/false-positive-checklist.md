# 误报排除检查表

## 通用检查

- [ ] 是否确认产品和版本受影响。
- [ ] 是否确认修复版本。
- [ ] 是否确认配置条件。
- [ ] 是否确认权限条件。
- [ ] 是否确认部署方式。
- [ ] 是否确认不是测试代码、README、示例配置或注释。
- [ ] 是否确认不是工具规则误报。
- [ ] 是否确认不是异常响应或错误页面。
- [ ] 是否确认不是信息暴露但无实际安全影响。
- [ ] 是否有代码证据。
- [ ] 是否有动态证据。
- [ ] 是否有影响证据。
- [ ] 是否有修复建议。
- [ ] 是否三次稳定复现。
- [ ] 是否检查 SRC 禁止边界。

## 结论

- 可报告：否
- 不能报告原因：
- 需要人工确认：是
