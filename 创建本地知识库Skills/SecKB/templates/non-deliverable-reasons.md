# 不可交付原因模板

用于证据不足、范围不明、规则冲突或质量门槛未通过时，阻止 Claude 硬交付低质量结果。

## 记录字段

- 任务 ID：
- 任务标题：
- 请求类型：学习 / 模板 / 验证 / 报告 / 审计
- 当前结论：cannot deliver / needs_review / conflict / stale / rejected
- 不可交付原因：
  - 缺少高可信来源
  - 缺少日期
  - 缺少版本或适用范围
  - 缺少影响证明
  - 缺少边界确认
  - 缺少误报排除
  - 缺少不可报告原因
  - 缺少修复建议
  - 缺少 evidence manifest
  - 仅有工具告警
  - 仅有错误页面
  - 仅有 README / 测试数据关键词
  - 命中 SRC 禁止边界
  - 目标未授权
  - 产品或版本不匹配
- 已有依据：
- 缺失依据：
- 安全下一步：
- 人工确认项：
