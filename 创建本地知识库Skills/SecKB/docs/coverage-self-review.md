## 主要风险

1. 真实环境无网络时，不能完成实时采集。
2. 各 SRC 规则变化快，需要人工确认官方页面。
3. 工具 release 解析可能因格式不同需要适配。
4. RAG 路由依赖索引质量。
5. 动态验证必须由用户提供明确授权范围。
